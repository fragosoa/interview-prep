# Interview Prep

A collection of competitive programming and interview problems solved in C++ and Python.

## Folder Structure

```
interview-prep/
├── cpp/                    # C++ solutions organized by platform
│   ├── codeforces/         # Codeforces competitive programming problems
│   ├── cses/               # CSES Problem Set (algorithmic problem set)
│   ├── leetcode/           # LeetCode interview problems
│   ├── usaco/              # USACO (USA Computing Olympiad) problems
│   ├── implementation/     # Data structures and algorithm implementations
│   └── misc/               # Miscellaneous problems from other sources
└── python/                 # Python reference scripts and LeetCode solutions
    └── leetcode/           # LeetCode problems solved in Python
```

## Problem Platforms

- **Codeforces** — competitive programming contests and practice problems
- **CSES** — structured algorithmic problem set covering core CS topics
- **LeetCode** — interview-focused problems (arrays, graphs, DP, trees, etc.)
- **USACO** — olympiad-style problems with a focus on implementation and algorithms
- **Misc** — problems from platforms like AlgoExpert and Meta coding assessments

## Topics Covered

- Arrays, strings, sliding window, two pointers
- Graphs: BFS, DFS, topological sort, DSU/Union-Find
- Trees: traversal, LCA, segment trees
- Dynamic programming (1D, 2D, interval, bitmask)
- Heaps, tries, monotonic stacks/queues
- Sorting and binary search
- Greedy algorithms
- Backtracking and recursion
- Data structures: LRU cache, min-stack, magic dictionary

## Python Reference

The [python/](python/) directory contains cheat sheets and standalone scripts for common data structures and patterns used in interviews (lists, heaps, dicts, sorting, slicing, OOP).
